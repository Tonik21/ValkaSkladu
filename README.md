### Valka Skladu
Tato hra počíná o Hráči, který je v dluzích a potřebuje si vydělat peníze na rodinu a proto jeho kamarád Richard Hangár ho pozval do Skladové říše
## Herní Funkce
- Pohyb Z lokace A do Lokace B
- Mluvení s Hernímy postavami
- Aukce
- Sebrání Itemů do Inventáře
## Herní Příkazy
**Pohyb**: Hráč se může hýbat z 
lokace do lokace
**Konec Hry** : Nečekaně ukončí hru

**Interakce** : Interakce se sklady a skladištěmi

**help** : Vytiskne vsechny commandy

**Talk** : Hráč si promluví s postavou v místnosti

**inventory** : Vytiskne Inventář

**status** : Aktualni status hráče(lokace, penize...)

**bid** : Hráč přihodí na aukci

**sell** : Hráč prodá Předmět v Inventáři

**take** : Hráč sebere Item z mistnosti

## Jak spustit
Stáhnout .zip
Extrahovat ho
spustit .jar pres cmd, najdete si ho ve např. file exploreru a pak v cmd upřesněte path a poté zadejte java -jar jackson-demo.jar
hra by měla být spuštěna
